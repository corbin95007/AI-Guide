AI 开发者技术栈参考矩阵

Architecture & Ecosystem Review (截止 2026-04-07)

LLM (Large Language Model)

大语言模型底层基础

科学原理

基于 Transformer 架构的自回归概率分布模型。通过在海量语料上进行无监督预训练，建立高维向量空间中的词汇关联网络。随后通过 SFT (监督微调) 与 RLHF (基于人类反馈的强化学习) 对齐指令。

作用机制

输入文本经过分词 (Tokenization) 转化为向量，通过自注意力机制 (Self-Attention) 并行计算上下文中各元素的依赖关系权重，自回归地计算并输出下一个最优 Token 的概率分布矩阵。

Agent (Autonomous Agent)

自治智能体

科学原理

具备环境感知、自主决策与物理/数字环境执行能力的计算实体。以 LLM 为核心逻辑控制器，辅以记忆系统（短期 Context 与长期 Vector DB）和推理框架（如 ReAct 或分层任务分解）。

作用机制

接收主干指令后，触发内部推理循环：评估当前状态 -> 规划步骤 -> 决定调用的外部工具接口 -> 解析工具返回结果 -> 更新上下文并迭代，直至达成终止条件。

MCP (Model Context Protocol)

模型上下文通信协议

科学原理

标准化的大语言模型与本地/外部数据源的双向通信层协议。通过统一定义文件系统、代码库、数据库的读取权限与工具执行规范，消除应用间的隔离壁垒。

作用机制

IDE 或客户端作为 Host，外部工具提供 MCP Server 暴露功能。当 LLM 需读取外部环境时，通过标准的 JSON-RPC 通信协议与服务器握手，授权并在安全沙箱内执行底层操作。

OpenClaw

跨平台自治 AI 代理网关与通信接口

科学原理

开源的 agent-native 网关服务，内置基于 `SKILL.md` 的技能编排树与工具调用鉴权模块。结合 x402 协议的微支付模块进行低延迟推理路由分发。

作用机制

以后台守护进程方式驻留，截获消息通讯协议（如 Telegram/Discord）的载荷，通过内置提示词工程解析用户意图，将指令分发至本地 Ollama 或云端 API，执行网络搜索或本地系统命令后回传结果。

Cursor

深度集成大语言模型的 AI 优先代码编辑器

科学原理

建立在 VS Code 开源分支上，集成嵌入级向量数据库（Vector DB）实现代码库级的上下文感知检索，并通过抽象语法树（AST）解析确保大模型输出的语法结构完整性。

作用机制

隐式捕获编辑器的光标位置、错误诊断信息（LSP Diagnostics）及相邻文件变更，将其打包作为 Prompt 发送至云端推理服务器，返回高精度的差异补丁（Diffs）并内联应用于源码。

Antigravity

基于 Agent 优先架构的异步协同 IDE (Google 发布)

科学原理

突破单行代码补全的范式限制，底层引入任务编排与状态机逻辑。依托 Gemini 3.1 Pro 模型的超长上下文窗口，实现对复杂工程的多代理（Multi-Agent）并行调度与环境隔离。

作用机制

用户通过“控制中心（Manager Surface）”发布高阶架构需求。系统实例化多个专职 Agent 并行执行（如编码、测试、文献阅读），通过生成严格的可验证构件（Artifacts：截图、执行日志、计划书）供人类异步审核。

Codex

生成式预训练代码 Transformer 模型

科学原理

OpenAI 开发的通用语言模型的代码微调分支，通过持续预训练（Continual Pre-training）海量公共代码仓库，深度拟合多种编程语言的抽象语法特征和逻辑分布。

作用机制

接收前置代码声明、文档注释作为输入序列条件，通过自回归解码机制计算后续可能代码块的联合概率分布，输出函数体、自动化测试或逻辑片段。

Ollama

本地大语言模型量化与运行时推理引擎

科学原理

基于 `llama.cpp` 核心实现 C/C++ 级的高效内存管理，采用 GGUF 格式对大模型权重进行动态降维量化（如 4-bit 量化），显著降低显存占用及内存带宽消耗。

作用机制

通过命令行 CLI 或 REST API 接收请求。将量化模型动态加载至系统 RAM 或 VRAM，通过调用底层硬件加速指令（如 CUDA, Metal, AVX）执行前向传播矩阵运算，返回流式 Token。

VSCode (Visual Studio Code)

可扩展跨平台源代码编辑器

科学原理

基于 Electron 架构与 Monaco 编辑器内核。核心技术为语言服务器协议 (LSP) 与调试适配器协议 (DAP) 的标准化解耦，将 UI 层与高 CPU 负载的语言解析层彻底分离。

作用机制

主进程控制生命周期与本地系统 I/O，渲染进程处理前端响应。所有的插件运行在独立的 Node.js 扩展宿主环境（Extension Host）中，确保主编辑器的帧率稳定与非阻塞输入。

OpenAI

GPT 家族及强化学习推理模型体系
o3 闭源

最新一代强化学习测试时推理（Test-Time Compute）模型，在顶尖算法与逻辑数学层面具备深度自我验证能力。

o1 闭源

初代引入链式思考（Chain of Thought）进行隐式内在推理的复杂任务解决模型。

GPT-4.5 闭源

旗舰通用模型版本，全面增强了超长上下文窗口的稳定性和系统级指令遵循精度。

GPT-4o 闭源

实现了全模态（文本、原生音频、实时视觉）端到端输入输出对齐的低延迟交互模型。

GPT-OSS-120B 开源

OpenAI 针对科研社区及高密级企业需求妥协释放的 120B 参数级开源变体模型。

Google

Gemini 多模态生态及 Gemma 开放权重体系
Gemini 3.1 Pro 闭源

原生支持百万级上下文，为 Antigravity 平台提供驱动的核心大模型，专精于多智能体异步协同调度。

Gemini 3 Flash 闭源

重点优化 TTFT（首字节延迟）与多模态流式输出速率的模型，适用于高并发的低延迟场景。

Gemini 2.0 Flash 闭源

2024 年末主打的性能平衡版本，具备稳健的原生函数调用（Function Calling）可靠性。

Gemma 3 27B 开源

基于 Gemini 同源架构训练的开放权重模型，适合通过本地显存矩阵或私有化服务器集群部署。

Gemma 3 9B 开源

微缩级参数量的端侧模型，针对代码补全辅助与离线基础语义解析进行了专项量化优化。

Anthropic

Claude 宪法人工智能及安全模型矩阵
Claude 4.6 Opus 闭源

2026年最新超大参数旗舰，解决高阶逻辑架构、复杂 API 集成及多步战略规划的最强变体。

Claude 4.6 Sonnet 闭源

在代码生成、AST 重构领域速度与精度的完美平衡，是 Cursor 等工具的首选驱动引擎。

Claude 4.0 Haiku 闭源

针对大规模并行处理、轻量级文本提取及 OCR 扫描任务优化的极速轻量模型。

Claude 3.5 Sonnet 闭源

前代在软件工程领域封神的里程碑式版本，被业界广泛集成于自动代码审查流水线。

Claude 3.5 Haiku 闭源

前代极具成本效益的低延迟变体模型。(注：Anthropic 核心政策目前未开放完全开源底层权重)。

DeepSeek (深度求索)

高性能开源混合专家 (MoE) 架构及专项微调模型
DeepSeek-V3 开源

671B 混合专家架构巨兽，采用无辅助损失负载均衡机制与 FP8 混合精度训练，打破闭源垄断的里程碑。

DeepSeek-R1 开源

通过大规模强化学习 (RL) 挖掘出系统潜能的推理模型，完全开源其链式思考机制与奖励模型细节。

DeepSeek-Coder-V2 开源

专门针对跨文件库上下文关联及 80+ 种主流编程语言范式微调的垂直领域专家模型。

DeepSeek-VL2 开源

多模态视觉语言模型，适用于复杂的前端 UI 蓝图反解、图表数据抽取任务。

DeepSeek-Math 开源

针对高等数学定理证明、LaTeX 公式推演与符号计算环境优化的专用小参数变体。
