AI 小白超车指南

一网打尽：核心概念 × 主流工具 × 大模型生态

🧠

LLM (大语言模型)

本质：AI 的“大脑”

阅读了人类几乎所有文本后训练出的超级文字处理器。它不是搜索引擎，而是通过“概率预测”来接话茬的专家。

🤖

Agent (智能体)

本质：有手有脚的“数字员工”

如果 LLM 是大脑，Agent 就是装上了四肢的打工人，能自己思考目标，使用工具去完成复杂任务。

🗣️

Prompt (提示词)

本质：给 AI 下达的指令

你对 AI 说的话。指令越具体、背景越清晰，结果就越好。写提示词就像给外星人布置工作。

🧩

Token (词元)

本质：AI 的基本颗粒

AI 眼里的字不是字，而是 Token。一个英文单词通常是 1 个 Token，一个汉字大约占 1-2 个 Token。

📝

Context (上下文)

本质：短期记忆容量

当前对话中喂给 AI 的背景信息。一旦超出了它的“窗口”，AI 就会像鱼一样忘记前面说过的话。

🔌

API (接口)

本质：软件间的点菜员

通过代码直接调用 AI 能力的通道。有了 API，你可以把 AI 接入到自己的微信或软件中。

💬 ChatGPT (文字之王)

适用：所有人（职场人、学生）

✨ 优劣势
  • 综合能力最强，响应快，插件多。
  • 需稳定网络，免费版受限。
📥 怎么用
  • 网页端访问 chatgpt.com。
  • 移动端去美区商店下载正版。
⚠️ 避雷区
  • 警惕国内同名盗版付费APP。
  • 勿输入公司机密或个人隐私。

🎨 Midjourney (绘画大师)

适用：设计师、自媒体、电商

✨ 优劣势
  • 出图质量极高，艺术感强。
  • 需通过 Discord，英文提示词门槛高。
📥 怎么用
  • 下载 Discord 并加入服务器。
  • 频道内输入 /imagine 画图。
⚠️ 避雷区
  • 新手易在公共频道丢失图片。
  • 建议拉入个人专属服务器操作。

👑 OpenAI 家族 (GPT-4o)

优势：全球标杆，多模态（看图听音）最强。

代表作：GPT-4o, o1 (强化推理模型)

👑 Anthropic (Claude 3.5)

优势：代码能力强，写作风格最像人类。

代表作：Claude 3.5 Sonnet

👑 国产之光 (DeepSeek / Qwen)

优势：中文理解地道，价格便宜，国内免翻墙直连。

代表作：DeepSeek-V3, Qwen-Max

获取渠道主要分为：官网按月订阅、购买 API 接口通过客户端使用，以及使用本地硬件运行开源模型。
